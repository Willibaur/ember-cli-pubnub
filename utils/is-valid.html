Method isValid

Determine the validity of a JSON type.

Parameters:

  • value must be an Object.
    (Any valid JSON type including objects, arrays, strings, and numbers.)

Returns a Boolean
(True if JSON is valid, otherwise false.)

export default function isValid(value) { if (typeof value === 'object' || typeof value === 'string' || typeof value === 'number' ) { return true; } return false; }
